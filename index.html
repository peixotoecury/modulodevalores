Como funciona:
1. Informe a Data de Ajuizamento (base dos juros) e a Data-base (ex: hoje)
2. Para cada verba, informe a Data de Origem = data de vencimento/competência da verba
3. O sistema calcula: Correção IPCA-E (origem→data-base) + Juros SELIC/1% (ajuizamento→data-base)
💡 Importando cálculo do contador (PJe-Calc): o principal vem com o valor já corrigido até a Data de Liquidação. A Data de Origem é preenchida com essa data — assim correção e juros calculam só o período após a liquidação (atualização até hoje). Para recalcular do zero, altere o principal para o valor original e a data de origem para a data de vencimento de cada verba.
